# 控制语句

## if

```txt
if (表达式)
{
    语句组;
}
```

## if else

```txt
if (表达式)
{
    语句组1;
}
else
{
    语句组2；
}
```

## if if else ... else

```txt
if (表达式1)
{
    语句组1;
}
if else(表达式2)
{
    语句组2；
}
else
{
    语句组3；
}
```

## switch

```txt
switch(表达式)
{
    case 条件1:
        语句1;
        break;
    case 条件2:
        语句2;
        break;
    default:
        语句3;
}
```
